# Simple Calculator

A basic calculator built using HTML, CSS, and JavaScript.  
It performs simple arithmetic operations and is designed as a beginner practice project.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Clear / Reset function

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Enter numbers using the buttons.
2. Select an operator (+, −, ×, ÷).
3. Click the equals (=) button to see the result.
4. Use the clear button to reset the calculator.
